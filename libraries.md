# List of Open-Source Email Address Validation Libraries

| Name | Repository | Language | Verification | Disposable | Notes |
|------|------------|----------|:------------:|:----------:|-------|
| **validator.js** | [validatorjs/validator.js](https://github.com/validatorjs/validator.js) | JavaScript | | | Most popular JS validation lib (23k+ stars). `isEmail()` function with configurable options. |
| **isemail** | [skeggse/isemail](https://github.com/skeggse/isemail) | JavaScript | | | RFC 5321/5322 compliant. Port of PHP is_email. |
| **deep-email-validator** | [mfbx9da4/deep-email-validator](https://github.com/mfbx9da4/deep-email-validator) | TypeScript | ✓ | ✓ | Validates regex, typos, disposable, DNS, and SMTP. Node.js only. |
| **email-validator** | [manishsaraan/email-validator](https://github.com/manishsaraan/email-validator) | JavaScript | | | Simple, fast syntax validator. |
| **python-email-validator** | [JoshData/python-email-validator](https://github.com/JoshData/python-email-validator) | Python | ✓ | | De facto standard for Python (4M+ weekly downloads). RFC-compliant, IDNA support, MX checks. |
| **egulias/EmailValidator** | [egulias/EmailValidator](https://github.com/egulias/EmailValidator) | PHP | ✓ | | Validates against multiple RFCs. Supports DNS and spoof checking. |
| **Apache Commons Validator** | [apache/commons-validator](https://github.com/apache/commons-validator) | Java | | | Part of Apache Commons. `EmailValidator` class. |
| **Hibernate Validator** | [hibernate/hibernate-validator](https://github.com/hibernate/hibernate-validator) | Java | | | Reference implementation of Jakarta Bean Validation. `@Email` annotation. |
| **AfterShip/email-verifier** | [AfterShip/email-verifier](https://github.com/AfterShip/email-verifier) | Go | ✓ | ✓ | Syntax, MX, SMTP, disposable, and typo detection. |
| **truemail** | [truemail-rb/truemail](https://github.com/truemail-rb/truemail) | Ruby | ✓ | | Zero dependencies. Regex, DNS, SMTP validation. Also available in Go. |
| **valid_email2** | [micke/valid_email2](https://github.com/micke/valid_email2) | Ruby | ✓ | ✓ | ActiveModel validation. MX lookup, disposable blocklist. |
| **EmailValidation** | [jstedfast/EmailValidation](https://github.com/jstedfast/EmailValidation) | C# | | | Simple, correct .NET validator. RFC 6531 (internationalized) support. |
| **email_address** | [johnstonskj/rust-email_address](https://github.com/johnstonskj/rust-email_address) | Rust | | | RFC 5322 compliant newtype. Supports ASCII and UTF-8. |
| **check-if-email-exists** | [reacherhq/check-if-email-exists](https://github.com/reacherhq/check-if-email-exists) | Rust | ✓ | ✓ | Full verification without sending email. AGPL license. (Powers Reacher) |
| **validator** | [Keats/validator](https://github.com/Keats/validator) | Rust | | | Struct validation with `#[validate(email)]` derive macro. |
| **MailChecker** | [FGRibreau/mailchecker](https://github.com/FGRibreau/mailchecker) | Multi | | ✓ | Cross-language (JS, Python, PHP, Ruby, Go, Rust, Elixir). 55k+ disposable domains. |
